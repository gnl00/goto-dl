### 偏移量

将 theta 变成 -b 表示偏移量

![image-20231029105555856](./assets/image-20231029105555856.png)

偏移量和权重的作用是不同的：

* 权重控制输入信号的重要程度
* 偏移量则最终定义了神经元的激活难易程度

### 阶跃函数

> 阶跃函数是指一旦输入超过阈值，就切换输出的函数。

函数 h 是一个阶跃函数

![image-20231029134709273](./assets/image-20231029134709273.png)

一旦输出超过阈值就会切换输出。

也就是说：上面的感知机使用阶跃函数作为激活函数。
