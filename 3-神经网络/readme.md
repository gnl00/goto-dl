## 阶跃函数

> 阶跃函数是指一旦输入超过阈值，就切换输出的函数。

函数 h 是一个阶跃函数

![image-20231029134709273](./assets/image-20231029134709273.png)

一旦输出超过阈值就会切换输出。

也就是说：上面的感知机使用阶跃函数作为激活函数。

> 在众多激活函数中选择了阶跃函数来作为激活函数，如果选择了其他函数会怎么样？
